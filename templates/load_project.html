Load Project Folder

Import an existing project folder into the incident reporting system

Enter the full path to the project folder you want to load. The system will copy and analyze all files in the folder.
Back to Dashboard

Supported File Types

  • JSON files - Configuration files, data files
  • YAML files - Configuration files, metadata
  • CSV files - Data files, spreadsheets
  • Text files - Documentation, logs, README files

What Happens During Loading?

  1. The project folder is copied to our projects directory
  2. All files are analyzed for type and structure
  3. Metadata is extracted and stored
  4. A project entry is created for incident tracking
  5. You can then create incidents linked to specific files
